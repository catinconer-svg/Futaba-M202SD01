# Библиотека M202SD01LA для Arduino

Библиотека для работы с VFD модулем Futaba M202SD01LA.

## Особенности

- Поддержка последовательного (UART) и параллельного интерфейсов
- Управление яркостью (6 уровней)
- Пользовательские символы
- Форматированный вывод

## Пользовательские символы (ВАЖНО!)

Модуль поддерживает 4 пользовательских символа: UF0, UF1, UF2, UF3.

### Коды символов:
- UF0 = 0x1B (27)
- UF1 = 0x1C (28)
- UF2 = 0x1D (29)
- UF3 = 0x1E (30)

### Формат данных:
Каждый символ состоит из 5 байтов, которые кодируют матрицу 5×7 точек.
**Внимание**: формат упаковки битов НЕ стандартный!

Байты распределяют 35 бит (5×7) особым образом:
- Байт 1: точки (1,1)-(5,1) и (1,2)-(3,2)
- Байт 2: точки (4,2)-(5,2) и (1,3)-(5,3) и (1,4)
- Байт 3: точки (2,4)-(5,4) и (1,5)-(4,5)
- Байт 4: точки (5,5) и (1,6)-(5,6) и (1,7)-(2,7)
- Байт 5: точки (3,7)-(5,7), биты 4-0 должны быть 0

### Пример: символ "1" (вертикальная линия)
```cpp
uint8_t char_one[5] = {0x23, 0x08, 0x42, 0x11, 0xC0};
vfd.defineCustomChar(0, char_one);
vfd.write(0x1B); // Выводим UF0
